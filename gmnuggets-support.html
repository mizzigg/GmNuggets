About this app...

  • GmNuggets app is completely free, free of cost, free of ads, free of any in-app purchases.
  • GmNuggets is a sensitive data management app that enables the user to easily enter, securely store and quickly find data... like: passwords, pins, bank-info, recipes, ideas, diary-entries, etc.
  • GmNuggets also helps with strong password creation when utilizing the included flexible password generator.
  • GmNuggets makes it very easy to securely share data.
  • GmNuggets data is managed solely by the user. GmNuggets data is created, viewed, edited or removed solely by the user. GmNuggets data resides solely on the user’s device… unless, the user backs-up or shares the data to a target other than their own device.

In-App Help

Help can be accessed by touching and holding the area of interest

First Use

A new password must be provided upon initial installation. The password must have a minimum of six characters. Nuggets can now be added.

Nuggets

Select Nuggets

While on the main screen touch select, nuggets are now selectable... selected nuggets can be shared, duplicated, printed or removed.

Add Nuggets

While on the main screen touch add. Provide nugget name when prompted. Fields can now be added.

Edit Nugget Name

Touch nugget then touch nugget name to edit

Share Nuggets

Select nuggets and then touch share. Provide a sharing password (receiver will use this password to import shared nuggets). Follow options and prompts to share

Duplicate Nuggets

Select nuggets and then touch duplicate to duplicate selected nuggets.

Print Nuggets

Selected nuggets will be printed.

NOTE: Use this with caution! Make sure that the printer being used is nearby or in a private location! This feature is intended as a hard backup to be put away in a private place in case all else breaks.

Remove Nuggets

Select nuggets and then touch remove to remove selected.

Filter Nuggets

Type in filter phrase and touch the filter button. Touch the cancel to remove filter.

Filter can be applied to:

  • Nugget Name
  • Field Value
  • Create Date
  • Modify Date
  • Import Info

Sort Nuggets

Nuggets can be sorted ascending/descending by:

  • Name
  • Date (modify date)

Fields

Reorder Fields

While inside a nugget touch the reorder button then touch up/down to reorder fields

Select Fields

While inside a nugget touch the select button, fields are now selectable... selected fields can be removed.

Add Fields

While inside a nugget touch the add button. Provide field name when prompted and then fill in field content.

Remove Fields

Select fields and then touch remove to remove selected fields.

Edit Field Name

Touch the field name to edit

Date Patterns

This is a data entry convenience tool. Preformatted date patterns can be created and used when entering field data.

Example: If a date pattern of "Dec 5 2021" is desired then do the following:

  • Touch Add
  • Touch Month refresh (cycle through formats) and stop when the format of MMM is reached
  • Touch Month add to add MMM to the pattern (MMM).

    NOTE: when formats are added to the pattern they are made visible at the bottom. This area is also where the last format added can be removed or the whole pattern cleared.
  • Touch Delim refresh and stop when space is reached (empty rectangle)
  • Touch Delim add to add space to the pattern (MMM ).
  • Touch Day refresh and stop when the format of d is reached
  • Touch Day add to add d to the pattern (MMM d).
  • Touch Delim add to add space to the pattern (MMM d ).
  • Touch Year refresh and stop when the format of yyyy is reached.
  • Touch Year add to add yyyy to the pattern (MMM d yyyy).

Password Patterns

This is a data entry convenience tool. Preformatted password patterns can be created and used when entering field data.

If for example the desired password requires the following attributes:

  • 6 characters long
  • letters, with at least one upper case and one lower case
  • numbers
  • at least one symbol limited to "-", "!" or "?"

A pattern like "Za@99" would generate passwords that satisfy the requirment. To create this pattern do the following:

  • Clear the current pattern by touching the "x" in the pattern box
  • Touch Consonant refresh until "Z" is showing
  • Touch Consonant add to add Consonant to pattern (Z)
  • Touch Vowel refresh until "a" is showing
  • Touch Vowel add to add Vowel to pattern (Za)
  • Touch Symbol table to select - ! ?
  • Touch Symbol add to add Symbol to pattern (Za@)
  • Touch Number add to add Number to pattern (Za@9)
  • Touch Number add to add Number to pattern (Za@99)
  • Touch pattern add to save pattern

Backup

Provide a name (optional) and a password to create a backup

Restore

To restore a backup:

  • Select a backup to restore from (provided that there were backups made)
  • Provide the password that was used to create the backup
  • Choose if the existing data is to be overwritten. If this is chosen then a prompt is displayed to confirm choice.
  • Touch accept to restore or touch cancel to cancel
